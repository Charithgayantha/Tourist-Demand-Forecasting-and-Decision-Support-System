# Tourist Demand Forecasting and Decision Support System (DSS) — Kandy District

An AI-powered decision support and forecasting system built specifically for Kandy's hospitality and tourism sectors. The platform uses advanced machine learning models to predict weekly tourist arrivals and provides interactive tools for scenario simulation, weather impact analysis, and resource planning.

## 🚀 Key Features
- **Weekly Tourist Arrival Forecasts**: Machine learning predictions (using Random Forest and LSTM architectures) for upcoming weeks.
- **Resource Planner**: Dynamic tools to forecast hotel occupancy, food/beverage demand, and staffing needs based on incoming guest volume.
- **What-If Scenario Simulator**: Interactive playground to simulate shifts in marketing, local infrastructure, or travel restrictions.
- **Festival Forecaster**: Tailored predictions for major festivals like Esala Perahera.
- **Weather Impact Analysis**: Models the correlation between weather conditions and traveler behavior.

## 🛠️ Tech Stack
- **Frontend/Interactive UI**: Streamlit (Python)
- **Database & Auth**: Supabase
- **Machine Learning**: Scikit-Learn, TensorFlow/Keras, XGBoost
